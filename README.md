## Usage

Every manual is a standalone `.html` file with inline CSS — no build step, no dependencies. Open any file in a browser, or browse the hub at `index.html`. Each manual ends with a **Sources** section linking the verified Microsoft Learn pages it is grounded in, plus a **learning path** to go deeper.
